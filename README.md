# Project Overview

This project presents an interactive Excel dashboard built from a single marketing dataset to analyze customer behavior and churn patterns. The dashboard integrates data cleaning, and pivot-based analytics to transform raw customer records into actionable business insights.

# Data Sources

The Primary dataset used for this analysis is the 'Marketing_data.xlsx', containing detailed information about customer demographics, purchase behaviour, campaing performance etc. the data set was curated by Kaggle, the no 1 data science platform.

# Tools

The tool used for this analysis was majorly Excel, under which i used to perform power Queries, power pivot, DAx and the dashboard creation.

# Data Cleaning and Preperation

In the data preperation stage, we performed the following tasks; Data Loading and inspection Handling missing values Data Cleaning and formatting, changed data type, reordered columns, unpivoted colunns etc.
